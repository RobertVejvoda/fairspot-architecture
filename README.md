# FairSpot Architecture Repository

This repository publishes the FairSpot architecture documentation as a standalone Docsify site.

The content follows a lightweight TOGAF 10 style:

- ADM navigation and checklist
- Architecture vision, principles, requirements, constraints, and decisions
- Business, information systems, technology, security, governance, and migration views
- Architecture states, gap analysis, risk register, and architecture versioning
- Viewpoint catalog and diagram/source-evidence index

The main FairSpot product and business documentation remains in:

https://github.com/RobertVejvoda/fairspot
